A modern and responsive educational platform developed for National Integrated College (NIC), designed to showcase academic programs, student life, admissions, scholarships, and institutional excellence through a professional digital experience.

Built with modern frontend engineering, responsive UI architecture, and performance-focused web technologies to create a clean and engaging educational platform.

## 🌐 Live Website

https://nationalintegratedcollege.edu.np

## 🎯 Project Purpose

The National Integrated College website was created to establish a strong digital presence for a modern academic institution by combining professional frontend development with clean educational-focused UI/UX principles.

The platform highlights:
- Academic programs
- Student opportunities
- Scholarships
- Internship culture
- College facilities
- Institutional branding
- Responsive educational experiences

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/panteliza/NIC.git
```

Navigate to the project folder:

```bash
cd NIC
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
